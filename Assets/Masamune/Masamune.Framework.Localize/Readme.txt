
#---------------------------------------------------------------------------
#  Localize - Masamune Framework
#---------------------------------------------------------------------------

The Masamune Framework is an asset series that conveniently extends Unity.
This asset primarily adds the following features to Unity and UniryEditor:

1.Text Translation
It is possible to translate a given text contained in dictionary data according to the language environment in which it is executed.
uGUI and UIElements translation components are available for immediate use.

2.Translatable UIElements tags
Tags such as translatable UIElements buttons (LocalizeButton) and labels (LocalizeLabel) will be available.

3.Dictionary Management with Google SpreadSheet
Use Google SpreadSheet to manage dictionaries for translation.
Google SpreadSheet has a translation function called "GOOGLETRANSLATE"
=GOOGLETRANSLATE("text" , "en" , "ja").
It is very convenient because you can create a dictionary using only machine translation without having to prepare a translator.

For this asset, you can use LocalizeInfo (Assets/Masamune/Modules/unit.style.localize/Resources)
You can manage spreadsheets and download them into Unity.

4.Includes several fonts
Fonts for major languages such as Chinese and Japanese are included.
And with a dedicated component, we can apply fonts automatically.
You don't have to suffer from the pain of not being able to see the letters.

5. Common Masamune Framework Features
In addition to the above, other useful features are provided.
Please see the following page for details.
https://mathru.net/unity/2905/

Masamune FrameworkはUnityを便利に拡張するアセットシリーズです。
このアセットでは主に下記のような機能がUnity及びUniryEditorに追加されます:

1. 文字の翻訳
辞書データに入っている与えられたテキストを実行されている言語環境に応じて翻訳することができます。
uGUIやUIElementsに対応した翻訳コンポーネントが用意されているのですぐに利用することができます。

2. 翻訳可能なUIElementsタグ
翻訳可能なUIElementsのボタン（LocalizeButton）やラベル(LocalizeLabel)などのタグが利用できるようになります。

3. Google SpreadSheetを用いた辞書管理
Google SpreadSheetを用いて翻訳用の辞書を管理します。
Google SpreadSheetでは「GOOGLETRANSLATE」という翻訳用の関数が用意されており
=GOOGLETRANSLATE( "text", "en", "ja")とセルに入力するだけで簡単に機械翻訳を行う機能があります。
翻訳家を用意しなくても機械翻訳だけで辞書を作成することができるので非常に便利です。

このアセットではLocalizeInfo（Assets/Masamune/Modules/unity.style.localize/Resourcesにあります）を利用することで
スプレッドシートの管理やUnity内へのダウンロードを行うことができます。

4. いくつかのフォントを同梱
中国語や日本語といった主要な言語用のフォントを同梱しています。
専用のコンポーネントを利用すればフォントの適用も自動で行いますので
文字が表示されない苦痛に悩まされることもありません。

Masamune Framework 는 Unity 을 편리하게 확장 할 자산 시리즈입니다.
이 자산은 주로 다음과 같은 기능이 Unity 및 UniryEditor 에 추가됩니다 :

1. 문자 번역
사전 데이터에 들어있는 주어진 텍스트를 실행하는 언어 환경에 따라 번역 할 수 있습니다.
uGUI과 UIElements에 지역화 된 구성 요소가 포함되어 있기 때문에 즉시 사용할 수 있습니다.

2. 번역 가능한 UIElements 태그
번역 가능한 UIElements 버튼 (LocalizeButton) 과 라벨 (LocalizeLabel) 등의 태그를 이용할 수있게됩니다.

3. Google SpreadSheet를 이용한 사전 관리
Google SpreadSheet를 사용하여 번역 용 사전을 관리합니다.
Google SpreadSheet는 "GOOGLETRANSLATE"라는 번역을위한 함수가 준비되어 있으며
=GOOGLETRANSLATE( "text", "en", "ja")와 셀에 입력하는 것만으로 간단하게 기계 번역하는 기능이 있습니다.
번역가를 준비하지 않아도 기계 번역 것만으로 사전을 만들 수 있기 때문에 매우 편리합니다.

이 자산은 LocalizeInfo (Assets/Masamune/Modules/unity.style.localize/Resources 에 있습니다)를 이용하여
스프레드 시트의 관리 및 Unity 내에 다운로드를 할 수 있습니다.

4. 어떤 글꼴을 포함
중국어 나 일본어 등 주요 언어의 폰트를 포함하고 있습니다.
전용 구성 요소를 사용하면 글꼴의 적용도 자동으로 이루어 지므로
문자가 표시되지 않는 고통에 시달린다 수 없습니다.
